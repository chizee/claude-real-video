# crv 免費版 demo 影片分鏡（codex 5.6-sol 對審版，2026-07-19 深夜定稿）

素材：NASA VIPER 月球車著陸點導覽（公有領域，2:45）
原則：全程真實錄影，不做假畫面。免費版佔 7-8 成、Pro 2-3 成。

## 分鏡（目標 60-75 秒）

1. 【0-8s】問題鉤子（字卡）：LLM 看影片其實只讀字幕 → crv 讓它真的「看」
2. 【8-25s】免費版閉環（終端真實錄影，可加速）：
   - `pip install claude-real-video`（一行）
   - `crv nasa.mp4 --speakers --viewer`
   - 真實輸出捲動：132 幀去重、逐字稿含說話者標記、67.6 秒跑完
3. 【25-40s】免費版產出展示：frames 資料夾縮圖牆 + transcript.txt + lite viewer 操作
4. 【40-60s】Pro 段（同一支影片）：viewer 感知時間軸 51 事件，只點高說服力事件：
   - 00:45 畫面：3D 動畫／CGI (0.70)
   - 01:34 音樂：漸強 → 01:36 文字：Nobile Region (0.89)（多模態連擊）
   - 點事件影片跳秒（互動入鏡）
   - 語言切換：繁中 → EN → 繁中，2-4 秒帶過
   - ⚠️ 避開：00:28-00:33 殘缺 OCR、00:42-43 低信心誤判、01:10 敲鍵盤、02:38-41
5. 【60-75s】收尾 CTA（回到免費版）：
   - 主：「先用免費版，把任何影片變成 LLM 真正看得懂的素材」
   - 次：pip install claude-real-video ＋ GitHub repo
   - 小字：Pro 提供進階感知時間軸
